cartographie Monde-001

Chaque image constitue une fenêtre sur un lieu remarquable d’un univers, un point de vue vers ce monde scindé en différents biomes. Les Dessins issues des images sont tracés par une machine en utilisant un stylo plume.

Navigation

  • Cliquer une image pour ouvrir son détail.

  • Maintenir la molette + glisser pour se déplacer sur les axes X/Y.

  • Utiliser la molette pour zoomer autour du pointeur.
